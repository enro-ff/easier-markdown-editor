# easier-markdown-editor

## 项目介绍
本项目使用vite + react + ts 构建，是一个简易的编写markdown文档的网站，致力于提供简单直观的markdown编写体验，提供了所见即所得的预览模式与纯粹的代码编辑面板，支持六级标题，*斜体*，**加重**，~~删除线~~，[超链接](仓库网址)，![图片]()，代码块
